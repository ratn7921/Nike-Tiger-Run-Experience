# Nike Tiger Run Experience

A premium, high-performance Nike landing page featuring immersive scrollytelling and a Pinterest-style masonry product grid.

## 🚀 Features

- **Immersive Scrollytelling**: High-performance canvas-based sequence animations using GSAP and ScrollTrigger.
- **Ambient Canvas System**: Dual-canvas setup for background blur and depth of field effects.
- **Masonry Product Grid**: Pinterest-style organic grid layout with varying aspect ratios and premium card hover states.
- **Full-Screen Interaction**: Image segments with dynamic full-screen text overlays.
- **Micro-Animations**: Smooth scale transitions, lift effects, and custom easing for a "wow" first impression.
- **Responsive Design**: Mobile-first architecture that scales elegantly across devices.
- **Cinematic Visuals**: Curated high-res imagery and subtle film grain for a premium editorial texture.

## 🛠️ Technology Stack

- **HTML5 / CSS3**: Semantic structure and modern CSS (Flexbox, Grid, Multi-column Masonry).
- **JavaScript (ES6+)**: Custom OOP-based logic for animation control.
- **GSAP (GreenSock)**: Industry-standard animation library.
- **ScrollTrigger**: Advanced scroll-based interactions.
- **Lenis**: Smooth scrolling engine for consistent cross-browser experience.

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ratn7921/Nike-Tiger-Run-Experience.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Nike-Tiger-Run-Experience
   ```
3. Use a local server (like Live Server or `http-server`) to view the site:
   ```bash
   npx http-server
   ```
